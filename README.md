# YT Continue Later

Chrome-расширение, которое автоматически сохраняет позицию воспроизведения YouTube-видео и генерирует ссылку с таймкодом для продолжения просмотра на другом устройстве.

## Возможности

- Автоматическое сохранение позиции при паузе
- Генерация ссылки с точным таймкодом
- Список последних 50 сохранённых видео с превью
- Копирование ссылки в один клик
- Тёмная тема в стиле YouTube

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/AlexWT1/extantion-youtube-link-time.git
   ```

2. Откройте `chrome://extensions/`

3. Включите **Режим разработчика** (Developer mode)

4. Нажмите **Загрузить распакованное расширение** (Load unpacked)

5. Выберите клонированную папку

## Использование

1. Откройте любое видео на YouTube
2. Поставьте паузу — позиция сохранится автоматически
3. Нажмите на иконку расширения в панели инструментов
4. Нажмите **Копировать ссылку** для текущего видео или выберите из списка
5. Вставьте ссылку на другом устройстве — видео откроется с сохранённой позиции

## Структура проекта

```

```
